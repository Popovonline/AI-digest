# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-06

### Added

- Initial release
- 34 curated sources across 11 categories
- I x R x C scoring methodology (Impact x Relevance x Confidence)
- Three-tier classification: MUST READ, RELEVANT, SKIP
- Telegram delivery integration
- Deduplication via seen-urls.json
- Calibration tracking for feedback
- OpenClaw SKILL.md agent definition

### Scoring

- MUST READ: score >= 50
- RELEVANT: score 20-49
- SKIP: score < 20
